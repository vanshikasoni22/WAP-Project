🎟️ BookMyShow Clone

A full-stack web application inspired by BookMyShow, built as a class project. This platform allows users to browse movies, view showtimes, select seats, and book tickets seamlessly.

🚀 Features
🎬 Browse Movies – Explore currently available movies with posters, ratings, and descriptions
🕒 Showtimes – View available show timings for different theaters
🪑 Seat Selection – Interactive seat layout with real-time availability
🧾 Ticket Booking – Book tickets and view booking summary
🔐 User Authentication – Sign up, login, and manage user sessions
📱 Responsive Design – Works across desktop and mobile devices
🛠️ Tech Stack

Frontend:

HTML, CSS, JavaScript
